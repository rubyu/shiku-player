## How to run the script
```bat
python shinku_player.py --path=path_to_いろとりどりのセカイ
```
and, then open http://127.0.0.0.1:5000 with your browser.

## Requirements

- Python 2.7
- iro-toridori-no-sekai (first edition); This software supports only the first version(1.0) of World.hcb.
